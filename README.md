Student Course and Performance Tracker

Overview
A command-line application built with Python and PostgreSQL to manage and track student academic performance for a single semester. It supports role-based access for students, teachers, and admins.

Features
Students: Login, add/drop courses, and view personal performance analytics.
Teachers: Import student performance records (via CSV) and view performance for their own courses.
Admins: Approve course enrollments, manage user data, and view overall analytics.

Tech Stack
Language: Python
Database: PostgreSQL
Interface: Command-Line (menu or command-driven)

Future Scope
Multi-semester support
Export reports to external files
Deeper analytics and visualizations
